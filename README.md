## ⚙️ Tech Stack & Architecture

| Layer | Technology |
|---|---|
| **Framework** | React 19 + TypeScript |
| **Build Tool** | Vite 7 |
| **Styling** | Tailwind CSS 4 |
| **Icons** | Lucide React |
| **Utility Libraries** | clsx, tailwind-merge |
| **Linting** | ESLint 9 + TypeScript ESLint |

### Design Highlights

- 🌑 **Dark Mode** theme with a sleek slate-900 base
- 🔮 **Glassmorphism** effects (backdrop blur + semi-transparent cards)
- ✨ **Scroll-reveal animations** on every section via custom `useScrollReveal` hook
- 📱 **Fully responsive** — mobile-first layout adapting to all screen sizes
- 🎨 **Gradient accents** and micro-animations on hover interactions

## 🚀 Getting Started

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/Koushikbm07/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
npm run preview
```
